## Built with Modern Android Technologies

TecNM Control Escolar is built entirely in **Kotlin** with **Jetpack Compose**, leveraging Material Design 3 for a beautiful, native Android experience.

<CardGroup cols={3}>
  <Card title="Jetpack Compose" icon="layer-group">
    Modern declarative UI toolkit for building native Android interfaces
  </Card>
  <Card title="Material Design 3" icon="palette">
    Latest design system with dynamic theming and adaptive layouts
  </Card>
  <Card title="Navigation Compose" icon="route">
    Type-safe navigation between screens with smooth transitions
  </Card>
</CardGroup>

## Quick Start

<Steps>
  <Step title="Install Android Studio">
    Download Android Studio Iguana or later from the official Android developer site
  </Step>
  <Step title="Clone the repository">
    Clone the project from GitHub and open it in Android Studio
  </Step>
  <Step title="Run the app">
    Start the Android emulator or connect a physical device and hit Run
  </Step>
</Steps>

<Note>
The app currently uses fake data for development. Supabase integration for backend connectivity is planned for future releases.
</Note>
